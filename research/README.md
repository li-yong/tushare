# research/

一次性实证研究脚本(手工运行,不进 cron,与根目录策略文件分开)。数据源 yfinance,运行需联网。

- `regime_confirm.py` — SPY/QQQ 熊市/牛市"确认阈值"实证:回撤深度/离峰天数 → 成熊概率;熊内反弹阈值 → 底部确认假信号率;对照 200 日线破线点。
- `ma_sensitivity.py` — MA 长度敏感性扫描(20~300 日 + 200 小时线对照):199 vs 200 是噪声,150~250 是平台,小时线鞭打 14~22 倍不可用。
- `qqq_100d.py` — QQQ 100 日线凹坑拆解:MaxDD 窗口/分年收益/逐段持仓盈亏(胜率 25%,靠长尾)。
- `qqq_50_100_verify.py` — 验证 QQQ 50/100 日线是否稳健优于 200 日:成本敏感性/分时段/剔除 2000-02/滚动 5 年胜率。结论:100 日淘汰;50 日=高保费早赔付的另一份保单,优势全在长熊,滚动窗口 59% 时间跑输。

- `hmm_regime_verify.py` — 高斯 HMM 体制识别验证(K=2/3,对数收益率):全样本平滑"前视版" vs 严格 walk-forward(季度扩展窗重拟合 + forward 滤波,无 backward)。结论:前视幻觉 ≈ +8.5pp CAGR;walk-forward 后 SPY 全窗/2003+窗都输 MA200 且鞭打 2 倍,QQQ 2003+ 赢 MA200(15.3% vs 12.1%,切换频率相当,熊离场更早)——但同窗 SPY 证伪了普适性,QQQ 结果按 n=1 对待;EM 对种子完全稳定。

结论口径:2026-07,QQQ/SPY 全历史日线,收盘口径,无执行摩擦(除 verify 脚本的 bps 档)。
