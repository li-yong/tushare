## 一、交易反什么人性 — 反的是为"确定性小世界"优化的快系统

人类学习机制为确定性环境进化(碰火即烫), 三个内核在市场里反向有害:

1. **单次学习失灵**(最深一层)。市场单次结果几乎全是噪声, 好决策亏钱/坏决策赚钱
   是常态; 人的学习机器从噪声里学出错误规律, 且亏一次大的比十份回测印象深。
   → 系统的对策: registry 认识等级制(huice=假说/ledger=知识/n≥10 才读 EV)
   = 给自己装慢速学习器, 压制出厂的快速学习器。
2. **高胜率偏好 vs 右偏收益结构**。采集环境奖励频繁小收益; 市场支付结构是
   "错很多次、对几次大的"(收益集中实证/huice 期望全靠右尾)。处置效应恰好反着做:
   剪右尾、留左尾。止损难受的本质: 把频繁的小失败感当成本预算 — 情绪账永远亏,
   只在期望账上赚。
3. **负反馈直觉 vs 正反馈市场**。自然界大多均值回归, "涨多该跌"是直觉;
   趋势市是正反馈系统 — 讨债单思维的进化论来源。

次级两条: 从众(正确姿态是对人群无感, 而人做不到)与行动偏好(系统大部分时间的
正确动作是什么都不做)。

**解法不是意志力**(消耗品, 实时对抗必输), 是奥德修斯式的: 清醒时把自己绑上桅杆
— 纪律外化成机制(分层止损程序化/ledger 自动盖章/⏳预算标注/止损尸检/提示 vs
门控分级)。适合交易的不是圣人, 是肯给自己设计约束的工程师。本 repo 与其说是
选股工具, 不如说是一台反 akrasia 机器。

## 二、适合交易的 mindset 五条

1. **身份绑规则不绑仓位**: 单笔是分布抽样, 评价对象永远是"规则跑遍 cohort 的 EV"
   (NVDA-2002: 牌赢了, 规则是输的 — docs/deep_drawdown_cohort_study.md)。
2. **知道哪层允许有观点**: 潮=服从/风=判断/浪=机械。失败模式=层错位
   (对潮行使判断"这次不一样", 对浪寄托信仰)。
3. **生存优先的不对称**: 复利是几何的, -90% 要 +900% 回来; 防归零优先于多赚,
   为小概率事件持续付保费(牛市里的假警报密度就是保费)。
4. **亏损当收据不当伤口**: 每次止损回答"哪条假设被证伪"(尸检四选一) —
   唯一心理上可持续的亏损观; 做不到的人积累创伤, 终点是不敢开仓或报复交易。
5. **认识论诚实是承重墙**: 测试不在赚钱时的冷静, 在止损后能否不找借口填尸检、
   暴赚后能否承认 ledger 说是运气。registry 强制"已改/明确不改"收尾 =
   把诚实变成程序, 不留给心情。

注: 4/5 依赖人性(诚实/耐心/把痛苦转化为认识)。交易反的是快系统, 吃的是慢系统。

## 三、市场有规律吗 — 有, 但载体是分布和体制, 且有半衰期

与物理规律的三点本质不同: **概率性**(分布上成立, 个案随时打脸)、
**体制条件性**(2022 全信号 meanR 转负 = 前提变了不是规律错了 → 规律必须带
有效域一起记)、**反身性**(被广泛利用即衰减 → 前向 ledger 高于回测的认识论根据)。
对规律的正确姿态 = 永不停机的 实践→认识→再实践 循环; 唯一不变的规律是
"必须持续检验规律"。

## 四、机会频率的梯子（量化, 2026-07-19）

| 层 | 是什么 | 频率(实测) |
|---|---|---|
| 信号 | 扫描器候选 episode | 每周 ~24 个 (ledger 2026-07 三周实测) |
| 合格 Setup | 过体制/质量门 | 每周几个 (MIXED 期为零) |
| 自我证明的机会 | 入场5天已涨>10% (huice 唯一正期望桶) | 约每月一两个 |
| 指数扛收益日 | 全年收益实际由几天贡献 | **中位 11 天/年 (4.3% 交易日)**; 错过最好5天, 年收益中位 +15.5%→+3.7% (^IXIC 1971~, 上涨年份 n=41) |
| 体制级 | 新鲜恐慌加仓窗 | 0.4 次/年 |
| 周期级 | SOX -40% 型板块深坑 | 每 5-6 年 |
| 世代级 | NDX-2000 型出清废墟 | 一代人一次 |

四条推论:

1. "日日有机会"对信号层成立, 对收益层不成立: **96% 的交易日在洗牌, 4% 在发钱**。
2. 交易者的工作不是找机会, 是**在场**(系统日日开机, 发钱日持有对的票)+
   **快速证伪**(时间预算/分层止损把假机会用最小代价送走 — 机会稀缺的镜像是
   占着假机会的成本极高)。
3. **频率与确定性严格反向**: 周频信号胜率平庸, 新鲜恐慌 f252 +33%, 指数深坑
   +1944% 近乎必然 → 恐慌加仓次序的时间维度版: 越稀有的机会越值得大仓位,
   越常见的信号越只配标准 R。多数人做反。
4. 脚注: "错过最好5天"有误导 — 最好单日 85% 聚集在 -20% 以深(§七),
   趋势系统本来就不在那些日子满仓; 错过部分最好日换躲开最坏月份,
   是 20 周线体系早已定价的交换, 不是缺陷。

## 五、是否应该一直满仓 — 仓位是输出, 不是输入

1. **"在场"指系统在场, 不是资金在场**: 在场的单位是"对的仓位"(自证仓位占比 +
   Open Heat, 不是资金利用率); 满仓横盘票 = 缺席的昂贵形式(时间预算实证)。
2. **几何复利惩罚恒定满仓**: 方差是税, Kelly 最优 < 全仓; 本组合是单一高相关主题
   (Open Heat 词条), 满仓五只半导体 = 一注乘五; SOX 每 5-6 年 -40% 的考试,
   几何复利不原谅缺考。
3. **恒定满仓在恐慌里换边**: 被迫卖出者正是"满仓+无缓冲"的人 — 新鲜恐慌来时
   非但无弹药执行加仓次序, 反而在供给恐慌。分层止损的第二功能: 下跌途中逐层
   砍出的现金, 恰好在恐慌标尺触发时出现在手上。
4. **反面错误同样致命**: 囤现金等崩盘 = 市场级讨债单(0.4次/年撑不起大额常备现金,
   STRONG 里空仓错过的是 11 天/年的工资); overlay 是公平保险不是收益增强, 反之
   现金只是保费不是收益来源。

**合成: 仓位 = f(信号供给, 体制, Open Heat), 让暴露呼吸** — STRONG+自证多→自然
近满仓; MIXED→不开新仓自然放气; WEAK/DEFEND→自然低仓; 新鲜恐慌→止损腾出的现金
按次序回填; 唯一硬约束 Open Heat ≤6%。

例外(可辩护的另一个游戏): **恒定满仓指数** = 服从潮的极端形式, 前提是期限>15年 +
扛得住 -80% + 中途永不被迫卖(NDX -70%处 +1944% 证明期望是真的, 14.6 年收复期
证明前提几乎没人满足)。若配置, 与深坑彩票仓同规: **单独一本账, 与 swing 账本
永不混记** — swing 的每条规则在那本账里都是错的, 反之亦然。混账是大多数人真正的
死因: 拿着长期账的理由, 扛着短期账的仓位。
